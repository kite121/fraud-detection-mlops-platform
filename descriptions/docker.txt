This module defines the containerization layer of the ingestion service using Docker and Docker Compose, enabling all system components to run together in a consistent environment. The Dockerfile is designed to build a lightweight Python-based service with all required dependencies and to run the FastAPI application via Uvicorn. The docker-compose configuration orchestrates multiple services — PostgreSQL, MinIO, and the ingestion-service — and connects them through a shared network and environment variables. A key design decision is to isolate responsibilities between services while ensuring seamless communication via service names, which makes the system scalable, reproducible, and easy to deploy.
